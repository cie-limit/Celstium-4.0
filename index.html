CELESTIUM 4.0 — QUICK GUIDE

What this is

Physics-based solar system simulator. Newton's law of gravitation, RK4 integration, Tsiolkovsky for mass depletion. No fake multipliers in any calculation.

Quick start

  • Choose MISSION PRESET, click APPLY PRESET, then PLAY.
  • WARP accelerates time. CAMERA changes view target.
  • MANEUVER: select direction, set ΔV, hit EXECUTE BURN.
  • ENABLE AUTO-LAND when near target body for GA landing.
  • Mission Data Link (bottom right) shows all events in chronological order.

Honest physics

  • N-body Newtonian gravity from Sun + Mercury + Venus + Earth + Moon + Mars.
  • Spacecraft initial state includes parent body's orbital velocity (vital for stable orbits).
  • Tsiolkovsky: ṁ = T/(Isp·g₀). No artificial scaling.
  • GA is honest: each run starts fresh, evaluates against current ship state, runs until you stop or max generations.

이 프로그램은

뉴턴 만유인력 + RK4 적분 + 치올콥스키 질량변화 기반의 정직한 태양계 시뮬레이터입니다. 결과를 조작하는 인위적 상수는 일체 없습니다.

빠른 시작

  • MISSION PRESET 선택 → APPLY PRESET → PLAY
  • WARP으로 시간 가속, CAMERA로 시점 변경
  • MANEUVER: 방향 선택, ΔV 입력, EXECUTE BURN
  • 목표 천체 근처에서 ENABLE AUTO-LAND로 유전 알고리즘 학습 시작
  • 오른쪽 아래 데이터 링크에 모든 이벤트가 시간순으로 기록됨

정직한 물리

  • 태양 + 수성 + 금성 + 지구 + 달 + 화성의 N체 뉴턴 중력
  • 우주선 초기 속도에 모천체의 공전 속도를 포함 (안정 궤도 필수)
  • 치올콥스키 ṁ = T/(Isp·g₀), 인위적 보정 없음
  • GA는 매번 새로 시작, 사용자가 STOP하거나 최대 세대 도달까지 진행
